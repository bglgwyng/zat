# CLAUDE.md

## Project Overview

`zat` is a CLI outline viewer for LLMs. It dispatches to language-specific viewers based on file extension patterns, configured via Nix. For directories, it looks for entry files and shows their outlines.

## Build

This is a Nix-only project (shell script generated by `flake.nix`). No Cargo.

```bash
nix build        # Build the package
nix flake check  # Validate
```

## Architecture

- **flake.nix**: The entire project. Generates a shell script that:
  1. Checks if input is a directory → looks for entry files, outputs their outlines
  2. Matches file extension against rules → dispatches to the appropriate viewer
  3. Falls back to `head -n 20` for unknown file types

- **Viewers** (separate repos):
  - `zat-js-viewer` — JS/TS (oxc parser)
  - `zat-rust-viewer` — Rust (syn parser)
  - `zat-python-viewer` — Python (regex-based)

All configurable via `makeOverridable`: rules, fallback, directoryIndex.
